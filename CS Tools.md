# 🛡️ Cybersecurity Toolkit Collection

A curated collection of widely used cybersecurity tools categorized by their primary purpose. This guide is useful for penetration testers, SOC analysts, digital forensic investigators, security researchers, and students learning cybersecurity.

---

# 🌐 Web Application Security

Tools used to identify vulnerabilities, intercept traffic, and assess the security of web applications.

| Tool | Purpose |
|------|---------|
| **Burp Suite** | Intercept, modify, and test HTTP/HTTPS requests for web security assessments. |
| **OWASP ZAP** | Free and open-source proxy for discovering web application vulnerabilities. |
| **SQLmap** | Automates SQL Injection detection and database exploitation. |
| **Nikto** | Scans web servers for outdated software, misconfigurations, and known vulnerabilities. |
| **Fiddler** | Captures and analyzes HTTP and HTTPS traffic. |
| **Wappalyzer** | Detects programming languages, frameworks, CMS platforms, and web technologies. |
| **Nessus** | Performs vulnerability assessments on web applications and infrastructure. |
| **Acunetix** | Automated scanner focused on web application security testing. |
